    • Here I host all of my projects, scripts, gists, dotfiles and this site too. Have a peek.

    • This learning app hopes to show how to read and pronounce basic Korean characters. It uses a tabbed layout with fragments and triggers sounds with a grid of onClickListeners. The row of tabs presents groups of consonants and the rows within the tabs show the most common consonant/vowel combination. The user presses each character to get a corresponding audio response in Korean.

    • This is a simple video recording app that uses no permissions. I used savedInstanceStates, onActivityResults and Toast messages. The aim was to make a quick loading video app that got out of the way of the user. It has very positive reviews, with some terrible ones too from younger Android users.

    • This is a fun app for children meant to emulate a lightsaber from the Starwars movie series. It uses sensorEvenListeners, shakeDetectors, and some other sensorManager imports. The user moves the phone around like a lightsaber and the app vibrates when sounds trigger, emulating a lightsaber fight.

    • Monster scanner is a silly app that I thought would be fun for really young children and first time dads/moms. It "scans" an area by taking a photo and then "detects" monsters in that area. It then returns a Toast message displaying if monsters have been found.
